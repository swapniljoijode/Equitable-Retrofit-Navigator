# CLAUDE Global Project Context

This repository builds the Equitable Retrofit Navigator for KC3.

## Non-Negotiables

- Never hallucinate legal, engineering, or financing numbers.
- Always return traceable citations for law references and incentives.
- Ask for human input when data quality is insufficient.

## Product Outcomes

- Reduce LL97 compliance risk for affordable housing.
- Improve resident comfort and affordability.
- Keep recommendations transparent and explainable.

## Engineering Defaults

- Python 3.11+.
- LangGraph for orchestration.
- Pydantic for node I/O contracts.
- ChromaDB + LlamaIndex for RAG expansion in future waves.
