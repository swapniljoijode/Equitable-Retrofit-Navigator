# PRD: Equitable Retrofit Navigator

## Problem

Affordable housing operators in NYC struggle to evaluate retrofit options that satisfy LL97 while keeping resident costs manageable.

## Users

- KC3 consultants
- Affordable housing operators
- Program managers coordinating grants and compliance

## Phase 1 Deliverable

An agentic planning engine that:

1. Evaluates LL97 compliance risk.
2. Validates available data.
3. Identifies likely incentives.
4. Generates retrofit plans.
5. Enforces affordability threshold (15% utility bill reduction target).

## Success Criteria

- Produces traceable structured output.
- Includes legal/program citations.
- Requests missing critical inputs from human users.
- Avoids unbounded planning loops.
